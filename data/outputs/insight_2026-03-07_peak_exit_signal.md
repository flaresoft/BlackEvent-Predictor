### 2.2 세 가지 법칙

**법칙 1: 고점이 높을수록 뒤따르는 충격이 크다**
- ~37 → -7~17% 낙폭
- ~64 → -18%+ 낙폭

고점이 높다 = 모든 영역에서 동시에 긴장이 쌓였다 = 트리거에 필요한 에너지가 적다.

**법칙 2: 고점이 높을수록 시간이 짧다**
- ~37 → 4개월의 여유
- ~64 → 3주의 여유

**법칙 3: 스코어가 떨어지는 구간이 가장 위험하다**

```
2/11: 63.7 (고점)    ← 마지막 탈출 시점
2/17: 52.3 (↓)
2/22: 36.1 (↓)      ← "괜찮아지나보다" (착각)
3/03: 26.0 (↓)      ← 이란 공습
3/04: -12% 폭락     ← 역대 최대
```

스코어 하락 = 안전해진 것이 아님.
뉴스가 지치고 있을 뿐, **구조적 취약성은 그대로인 채 트리거만 기다리는 구간**.
